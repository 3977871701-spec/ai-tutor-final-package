## 2. 安装Python

### Windows系统

1. 打开浏览器访问：https://www.python.org/downloads/windows/
2. 点击 "Download Python 3.11.x" 下载
3. 双击安装包运行
4. **重要**：勾选 "Add Python to PATH"（添加环境变量）
5. 点击 "Install Now"

### Mac系统

1. 打开浏览器访问：https://www.python.org/downloads/mac-osx/
2. 下载 Python 3.11
3. 双击安装包，按提示安装

或者使用Homebrew（推荐）：
```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
brew install python@3.11
```

### Linux系统

```bash
sudo apt-get update
sudo apt-get install python3 python3-pip
```

### 验证安装

打开命令行（Windows按Win+R，输入cmd回车），输入：

```bash
python --version
```

看到类似 `Python 3.11.x` 表示安装成功。

## 3. 安装依赖

### 方法一：一键安装（推荐）

1. 打开命令行，进入程序文件夹：
```bash
cd 桌面/AI辅导员
```

2. 运行安装脚本：
```bash
# Windows
python install_dependencies.py

# Mac/Linux
python3 install_dependencies.py
```

等待安装完成（可能需要5-10分钟）。

### 方法二：手动安装

如果一键安装失败，尝试手动安装：

```bash
# Windows
pip install -r requirements.txt

# Mac/Linux
pip3 install -r requirements.txt
```

## 4. 配置系统

### 4.1 获取微信公众号凭证

1. 登录微信公众平台：https://mp.weixin.qq.com/
2. 进入 "设置与开发" → "基本配置"
3. 记录以下信息：
   - AppID（应用ID）
   - AppSecret（应用密钥）

### 4.2 获取xAI API密钥（可选）

1. 访问 https://x.ai/api
2. 注册账号并获取API Key
3. 如果无法访问xAI，系统会自动使用本地知识库回答

### 4.3 编辑配置文件

用记事本打开 `config.yaml` 文件，修改以下内容：

```yaml
xai:
  api_key: "这里填入你的xAI密钥"  # 可选，没有可以留空

wechat:
  app_id: "wx1234567890abcdef"      # 填入你的AppID
  app_secret: "abcdef1234567890"   # 填入你的AppSecret
  token: "your_token_here"          # 填入你设置的Token
  encoding_aes_key: "your_aes_key"  # 填入你的AES Key

teacher:
  contacts:
    - name: "张老师"
      phone: "010-12345678"
      department: "学生工作办公室"
      specialty: "奖助学金,助学贷款"
    - name: "李老师"
      phone: "010-87654321"
      department: "宿舍管理科"
      specialty: "宿舍调整,住宿咨询"
```

**保存文件**（Ctrl+S）。

## 5. 启动服务

### Windows系统

**方法一：双击启动**
- 双击 `start.bat` 文件

**方法二：命令行启动**
```bash
cd 桌面/AI辅导员
python start.bat
```

### Mac/Linux系统

```bash
cd ~/Desktop/AI辅导员
chmod +x start.sh
./start.sh
```

### 验证服务运行

打开浏览器，访问：http://localhost:8000

看到 "AI辅导员系统已启动" 表示成功。

## 6. 配置微信公众号

### 6.1 配置服务器地址

1. 在微信公众平台，进入 "设置与开发" → "基本配置"
2. 点击 "修改配置"
3. 填写以下信息：

| 字段 | 填写内容 |
|------|----------|
| URL | `http://你的公网IP:8000/wechat` |
| Token | 与config.yaml中一致 |
| EncodingAESKey | 与config.yaml中一致 |
| 消息加密方式 | 选择"安全模式"（推荐）或"明文模式" |

### 6.2 使用ngrok进行本地开发测试

如果暂时没有公网服务器，可以用ngrok进行测试：

1. 下载ngrok：https://ngrok.com/download
2. 注册账号获取authtoken
3. 运行ngrok：
```bash
ngrok http 8000
```
4. 复制显示的 https://xxxx.ngrok.io 地址
5. 将 URL 填入微信公众号后台

### 6.3 验证配置

在微信公众号后台，点击"提交"按钮。

## 8. 日常维护

### 8.1 更新知识库

#### 方式一：命令行

```bash
# 查看当前状态
python manage_knowledge.py status

# 列出所有知识
python manage_knowledge.py list

# 添加新知识
python manage_knowledge.py add-scholarship -c "新奖学金内容"

# 导出备份
python manage_knowledge.py export -f backup.json
```

#### 方式二：API接口

启动API服务后，可以远程管理：

```bash
# 启动API服务（端口8001）
python import_api.py

# 添加网页内容
curl "http://localhost:8001/api/import/url?url=网址&category=scholarship"

# 上传文件
curl -X POST -F "file=@文档.pdf" http://localhost:8001/api/import/file
```

### 8.2 定时抓取学校公告

```bash
# 手动抓取
python school_news_fetcher.py

# 定时任务（需要配置crontab）
crontab -e
# 添加：0 8 * * * /path/to/python /path/to/school_news_fetcher.py
```

### 8.3 查看日志

日志文件在 `logs/` 文件夹下：

- `app.log` - 应用日志
- `error.log` - 错误日志

### 8.4 常见问题

**Q: 服务启动报错"端口被占用"**
```bash
# Windows查找占用
netstat -ano | findstr :8000

# Mac/Linux查找占用
lsof -i :8000
```

**Q: AI不回答，只显示知识库内容**
- 检查xAI API是否可用
- 检查网络连接

**Q: 微信公众号无法接收消息**
- 检查config.yaml中的AppID、AppSecret是否正确
- 检查微信公众号后台URL配置是否正确
- 确保防火墙允许8000端口
